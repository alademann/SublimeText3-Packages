Element Finder
==============

Thanks for installing Element Finder.

Please note that NodeJS is required to run Element Finder. If you do not have NodeJS installed, go to http://nodejs.org

If Element Finder still doesn't work, please follow these steps:
1. Go to Sublime Text 2 > Preferences > Package Settings > Element Finder > Settings - Default
2. Copy the default settings
3. Go to Sublime Text 2 > Preferences > Package Settings > Element Finder > Settings - User
4. Paste the settings
5. Change the "node_path" setting to point to NodeJS on your system
