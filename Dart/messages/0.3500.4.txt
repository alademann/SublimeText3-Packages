Wiki: https://github.com/dart-lang/dart-sublime-bundle/wiki


MAIN CHANGES
------------

	- Remove command to open DartEditor
	- Add command palette item to stop running scripts
	- Added additional args to Dartium launch

	Dartium should behave more similarly to what it does when run from the
	Dart Editor.
